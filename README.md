# diary-of-emotions-vk-bot
Интеллектуальный помощник - бот "ВКонтакте" для ведения дневника настроения и обучения техникам эмоциональной саморегуляции

Запускать по очереди:
1. data_processing.py
2. categorization.py
3. train.py
4. bot_essentials.py
5. bot.py



В файле bot.py заполнить:
VK_TOKEN = ''                               # Заполнить токен
GROUP_ID =                                  # Заполнить ID группы

Инструкция для получения токенов и ID размещена в файле "Инструкция для токенов VK"
